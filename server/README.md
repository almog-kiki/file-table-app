# Gallery server

## Versions
```
    npm -v 6.14.6
    node v12.18.3
```

# How to run
``` 
npm install 
npm start 
```

# Tests
```
npm jest
```

# APIs: 
## get files names
```
localhost:8080/api/file/names 

```

## get the conent of the file
```
localhost:8080/api/file/?file_name=a.json
```
